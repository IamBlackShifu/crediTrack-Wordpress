CrediTrack transactional lending and credit-management services.

== Installation ==

1. Upload and activate CrediTrack Core.
2. Install and activate the matching CrediTrack Portal theme.
3. Visit Settings > Permalinks and save once.
4. Open /creditrack/settings/ as an administrator.

== Upgrade Notice ==

Take a host-level database and uploads backup first. Update the core plugin before the portal theme, then open an authenticated CrediTrack page to run migrations and overdue refreshes.

== Security ==

Use HTTPS in production. Configure a real system cron, protected private storage, monitored email delivery, least-privilege database access, and host-level backups. This package does not itself establish regulatory compliance.

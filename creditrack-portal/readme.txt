=== CrediTrack Portal ===
Author: Infinity Lines of Code (Pvt) Ltd
Requires at least: 6.6
Requires PHP: 8.1
Version: 0.3.5

Responsive authenticated management portal for CrediTrack Core.

== Installation ==

Install and activate the matching version of CrediTrack Core first. Upload and activate this theme, then save WordPress permalinks once. Signed-in CrediTrack users enter at /creditrack/dashboard/.

== Accessibility and browser support ==

The portal uses semantic server-rendered forms, keyboard-visible focus, responsive tables, reduced-motion support, and dependency-free charts. Current evergreen browsers are recommended.
